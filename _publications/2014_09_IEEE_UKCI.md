excerpt: '[UKCI'14] Detecting dataset shifts in non-stationary environments is crucial for adaptive learning. This paper introduces an algorithm using an exponential weighted moving average (EWMA) model to detect shifts in time-series data, allowing timely adaptation of the classifier's knowledge-base. The algorithm, designed for real-time learning, exhibits effective reactions to various covariate shifts in synthetic datasets.'

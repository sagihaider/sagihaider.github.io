excerpt: '[PR'15] Detecting dataset shifts is challenging in non-stationary environments. This paper proposes novel methods for covariate shift-detection tests in univariate and multivariate time-series. The first stage, an exponentially weighted moving average (EWMA) model, detects shift points, while the second stage validates using statistical tests. The approach, suitable for real-time implementation, demonstrates improved performance in detecting covariate shifts with reduced false alarms.'
